# ZKO_20XX
A desktop app for Super Smash Bros. Melee players that visualizes game data and matchup statistics in real-time.

At the start of match ZKO dynamically loads data according to your character, opponent character, and stage selection.

## Install

**[Download the latest release](https://github.com/craigf-svg/ZKO20XX/releases)** for Windows, macOS, or Linux.

## Setup Guide

 **[ZKO20XX.gg](https://zko20xx.gg/start)**

## Build from Source

**Prerequisites:**
- Node.js (version 18 or higher)
- Rust (for Tauri desktop builds)

```bash
# Clone the repository
git clone https://github.com/craigf-svg/ZKO20XX.git
cd ZKO20XX

# Install dependencies
npm install

# Run in development mode
npm run tauri dev
```
